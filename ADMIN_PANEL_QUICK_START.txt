┌─────────────────────────────────────────────────────────────────────┐
│  HOW TO OPEN ADMIN PANEL                                            │
└─────────────────────────────────────────────────────────────────────┘

Simply type:  /admin

You'll see a beautiful menu with buttons! ✨


┌─────────────────────────────────────────────────────────────────────┐
│  MAIN MENU OPTIONS                                                  │
└─────────────────────────────────────────────────────────────────────┘

💰 POINTS MANAGEMENT
   └─ Add/remove points, view leaderboard

⭐ TRUSTED SELLERS  
   └─ Add/remove verified sellers

🚨 SCAMMER LIST
   └─ Manage scammer database

📋 REVIEW CLAIMS
   └─ Approve/reject scammer reports

🔍 USER LOOKUP
   └─ Check any user's complete profile

📊 STATISTICS
   └─ View system stats and health

⚙️ SETTINGS
   └─ Coming soon!

❌ CLOSE
   └─ Close the admin panel


┌─────────────────────────────────────────────────────────────────────┐
│  QUICK ACTIONS                                                      │
└─────────────────────────────────────────────────────────────────────┘

📌 GIVE POINTS TO USER:
   /admin → Points → Add Points → @username 100

📌 ADD TRUSTED SELLER:
   /admin → Sellers → Add Seller → @username

📌 REPORT SCAMMER:
   /admin → Scammers → Add → @username | reason

📌 REVIEW PENDING REPORTS:
   /admin → Review Claims → Select Report → Confirm/Dismiss

📌 LOOK UP USER:
   /admin → User Lookup → @username

📌 VIEW STATS:
   /admin → Statistics


┌─────────────────────────────────────────────────────────────────────┐
│  INPUT FORMATS                                                      │
└─────────────────────────────────────────────────────────────────────┘

Points:         @username 100
Seller:         @username
Scammer:        @username | Reason for report
User Lookup:    @username
